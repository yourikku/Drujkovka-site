# Сайт о Дружковке - История и Культура

Многостраничный информационный веб-сайт, посвящённый истории, культуре и инфраструктуре города Дружковка.

## Описание проекта

Этот проект создан в рамках лабораторной работы №1 по курсу "Web-программирование" (вариант 9). Сайт демонстрирует применение современных технологий HTML5 и CSS3 для создания полноценного информационного веб-ресурса.

## Использованные технологии

- **HTML5** - семантическая разметка, современные теги (nav, header, footer, section, article)
- **CSS3** - стилизация, адаптивный дизайн, градиенты, тени, анимации, transitions
- **JavaScript** - валидация формы обратной связи, интерактивность
- **Git/GitHub** - система контроля версий и хостинг кода

## Структура сайта

Сайт состоит из 6 основных страниц:

1. **index.html** - Главная страница с общей информацией о городе и фотогалереей (10 фото)
2. **about.html** - Страница о разработчике с биографией и диаграммой профессиональных навыков
3. **history.html** - История города с таблицей из 12 значимых исторических событий
4. **infrastructure.html** - Инфраструктура города (географическое положение, транспорт, коммунальные услуги, административное деление)
5. **culture.html** - Культура и искусство (театры, музеи, библиотеки, галереи, фестивали, известные деятели)
6. **contact.html** - Форма обратной связи с валидацией полей

## Технические требования

Проект полностью соответствует требованиям лабораторной работы:

### HTML:
- Использовано **55+ различных HTML-тегов**
- Применено **минимум 15 тегов HTML5**: nav, header, footer, section, article, main, aside, figure, figcaption, time, mark, details, summary и др.
- Все теги применены корректно с соблюдением семантики
- Никакие конструкторы сайтов не использовались

### CSS:
- **3 способа позиционирования**: static (по умолчанию), relative (для элементов), sticky (для навигации)
- **Минимум 1 способ обтекания**: используется для layout элементов
- **6+ видов простых селекторов**: по тегу, классу, ID, атрибуту, универсальный, групповой
- **2+ типа комбинаторов**: потомки (пробел), дочерние (>), соседние (+), общие соседние (~)
- **10+ псевдоклассов**: :hover, :active, :focus, :visited, :link, :first-child, :last-child, :nth-child(), :not() и др.
- **Минимум 1 псевдоэлемент**: :before, :after для декоративных элементов

## Дизайн и стилизация

Сайт оформлен в тёмно-синей цветовой гамме с акцентами:
- Основной цвет: #0f2027 (тёмно-синий)
- Акцентные цвета: #203a43, #2c5364
- Светлые акценты: #a8dadc
- Фон: градиент от тёмно-синего к голубоватому

Использованы:
- Градиентные фоны для создания глубины
- Box-shadows для объёмности элементов
- Transitions для плавных анимаций
- Border-radius для скругления углов
- Hover-эффекты для интерактивности
