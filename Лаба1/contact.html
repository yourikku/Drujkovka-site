О проекте

Этот сайт создан в образовательных целях для демонстрации навыков работы с HTML5 и CSS3. Проект включает использование семантической разметки, современных CSS-селекторов, адаптивного дизайна и валидации форм.

Сайт полностью функционален и может использоваться как портфолио работ по веб-разработке. Все материалы носят информационный характер.

Технологии

В проекте использованы следующие технологии и подходы:

• HTML5 с семантическими тегами

• CSS3 с градиентами, тенями и анимациями

• JavaScript для валидации форм

• Адаптивный дизайн

• Git для контроля версий
